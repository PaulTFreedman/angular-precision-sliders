This is a project that aims to improve upon the classic slider in situations where the range of data is very large. For example, the following slider has a range of 5000; try setting it to 3456. Difficult, no?
Now try doing the same with the precision slider below. Get the slider handle to roughly the desired value, then drag downwards. This will "focus" the range of the slider, making it easier to hit the correct value. (It's easier to understand if you just try it out!)

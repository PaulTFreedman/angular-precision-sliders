This is a project that aims to improve upon the classic slider in situations where the range of data is very large. For example, the following slider has a range of 2500; try setting it to 1234. Difficult, right?
Now try doing the same with the precision slider below. Get the slider handle to roughly the desired value, then drag downwards. This will "focus" the range of the slider, making it easier to hit the correct value. (It's easier to understand if you just try it out!)
Hopefully that was a lot easier. You can also change the focus rate if you want even more precision!

You can also have a non-responsive slider. Type a value between 0 and 2500 in the input below to update the value of the slider.
